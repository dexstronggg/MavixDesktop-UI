# MavixDesktop — Руководство оператора

Документ составлен по ГОСТ 19.505-79 «Руководство оператора» (ЕСПД).

## 1. Аннотация
Руководство оператора приложения MavixDesktop системы доставки грузов дронами Mavix.

## 2. Назначение программы
Приложение позволяет оператору принимать заявки на доставку, подключаться к дрону,
наблюдать видео, телеметрию и карту, управлять полётом и сбрасывать груз.

## 3. Условия выполнения программы
- ОС: Linux x64 / Windows 10+. Джойстик (USB). Интернет.
- Для MAVLink-полёта — установленный QGroundControl.
- Учётные данные оператора (логин/пароль) выдаёт администратор.

## 4. Выполнение программы

### 4.1 Вход
Запустить приложение, ввести логин и пароль оператора.

### 4.2 Приём заявки
На экране ожидания отображаются заявки на доставку. Нажать «Принять» —
кто принял первым, тот и ведёт доставку; приложение подключается к дрону.

### 4.3 Управление дроном
- Видео с камер (переключение стрелками ←/→), телеметрия (батарея, задержка),
  карта в углу (позиция дрона, поворот по курсу, точка назначения).
- **CRSF-полётник:** управление джойстиком из приложения.
- **MAVLink-полётник:** нажать «Взлёт» — приложение запустит QGroundControl с
  настройкой джойстика; полёт ведётся в QGC, ARM/DISARM дублируется приложением.
- Калибровка джойстика — на экране «Джойстики» (включая привязку кнопки сброса).

### 4.4 Сброс груза
По прибытии нажать кнопку сброса (на джойстике или на экране) — груз сбрасывается
(AUX-канал), заявка отмечается доставленной, администратор получает уведомление.

### 4.5 Завершение
Кнопка «Назад» возвращает к списку заявок и разрывает сессию с дроном.

## 5. Сообщения оператору
- «Закройте QGroundControl» — QGC уже запущен; закрыть и нажать «Взлёт» снова.
- «Ожидание GPS…» на карте — борт ещё не передал координаты (нет фикса спутников).
- «Контроллеры не найдены» — подключить джойстик (список обновится автоматически).
- Дрон ушёл офлайн → приложение вернёт к списку заявок.
