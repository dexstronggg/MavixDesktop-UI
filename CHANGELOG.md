# Changelog

Все значимые изменения станции оператора MavixDesktop документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект придерживается [семантического версионирования](https://semver.org/lang/ru/).

## [1.0.0] - 2026-06-08

Первый стабильный релиз станции оператора комплекса доставки грузов на базе БПЛА.

### Added
- Аутентификация и работа с сервером по REST, приём видео и каналов данных по WebRTC.
- Дистанционное пилотирование: формирование RC-кадров CRSF из положений стиков геймпада.
- Калибровка геймпада с сохранением профиля по модели устройства.
- Полётное окно: видеопоток, индикатор полётного контроллера, панель параметров, ARM/DISARM, сброс груза.
- Нативная карта на QPainter (без QtWebEngine), демо-карта до получения GPS-фикса.
- Скрипт запуска тестов (pytest, offscreen) и покрытие тестами до 72%.

### Fixed
- Устранено зависание окна при открытии QGroundControl.
- Быстрый hot-plug джойстика через `event.pump()`.
- Корректная работа windowed-сборки при `sys.stderr is None`.
